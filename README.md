# DEV

1. Clone the files .env.template to .env
2. Set up the environment variables
   ...
   PORT = 3000

   MAILER_EMAIL=

   MAILER_SECRET_KEY=

   PROD=false

3. Execute command **_npm install_**
4. Execute command **_npm run dev_**
